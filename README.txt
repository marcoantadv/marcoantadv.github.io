COMO VER O SITE NO COMPUTADOR
1. Abra a pasta deste projeto.
2. Clique duas vezes em index.html.
3. O site abrirá no navegador.

COMO PUBLICAR GRATUITAMENTE COM GITHUB PAGES
1. Crie uma conta em github.com.
2. Clique em “+” no canto superior direito e escolha “New repository”.
3. Dê ao repositório um nome simples, por exemplo: marco-antonio-advocacia.
4. Escolha “Public” e clique em “Create repository”.
5. Dentro do repositório, clique em “Add file” > “Upload files”.
6. Abra esta pasta descompactada e envie TODOS os ficheiros e a pasta “assets”.
   Importante: o ficheiro index.html e a pasta assets devem ficar na raiz do repositório.
7. Clique em “Commit changes”.
8. Abra “Settings” do repositório, depois “Pages”.
9. Em “Build and deployment”, selecione “Deploy from a branch”.
10. Escolha a branch “main”, a pasta “/(root)” e clique em “Save”.
11. Aguarde alguns minutos. O GitHub mostrará o endereço do seu site, normalmente:
    https://SEU-UTILIZADOR.github.io/marco-antonio-advocacia/

Para alterar o site depois, use “Add file” > “Upload files” para substituir index.html
ou imagens dentro da pasta assets. Após “Commit changes”, o GitHub atualiza o site.
